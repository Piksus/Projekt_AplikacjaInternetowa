# EventMeet - Instrukcja użytkownika

## Dla uczestnika (bez logowania)

1. Otrzymujesz od organizatora link do wydarzenia (mailem, na czacie itp.) lub po prostu
   znajdujesz je na liście wydarzeń.
2. Wejdź na **link do wydarzenia** (`/Events/Details/{id}`) - zobaczysz opis, datę,
   miejsce, zdjęcie, regulamin (jeśli dodany) oraz listę uczestników wraz z ich statusami
   (Idę / Może / Nie idę / Brak odpowiedzi).
3. Swoją decyzję (idę/nie idę/może) zgłaszasz organizatorowi w zwykły sposób
   (telefon, wiadomość, czat) - organizator wpisuje ją w panelu.
4. Nie musisz się rejestrować ani logować.

## Dla organizatora

1. Wejdź na `/Account/Login` i zaloguj się (login: `organizator`, hasło: `haslo123`).
2. W **Panelu organizatora** widzisz listę wszystkich wydarzeń.
3. **Dodaj wydarzenie** - kliknij "+ Dodaj wydarzenie", wypełnij nazwę, datę, miejsce,
   kategorię, opis, opcjonalnie dodaj zdjęcie i plik z regulaminem/informacjami.
4. **Edytuj** - zmień dane wydarzenia lub podmień zdjęcie/regulamin.
5. **Uczestnicy** - dodaj uczestników (imię i nazwisko + email). Gdy uczestnik zgłosi Ci
   swoją decyzję (np. telefonicznie), wybierz jego status z listy rozwijanej w tabeli
   (Idę / Może / Nie idę / Brak odpowiedzi) - zapisuje się automatycznie.
6. **Usuń** - usuwa wydarzenie razem z listą jego uczestników.
